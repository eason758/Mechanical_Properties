【下載】
    #只開放Windows下載#
->按中右的綠色按鈕(Code)
->點選Download ZIP
->ZIP解壓縮


【開啟】

->進入資料夾後找 "Mechanical Properties.exe"
->右鍵->以系統管理員身分執行


【使用】

->使用介面為命令提示字元(CMD, 會自動跳出)
->基本上他問啥就回答啥
->檔案格式須為.csv檔 且 第一列為單位 第一行為displacement,單位須為cm 第二行為stress,單位須為N
->完整路徑包含C槽/D槽 到 檔案名稱(包含副檔名.csv) 
  (可右鍵檔案->內容->位置 複製後再加"\"再加檔案名稱)
->YS data range 是指你想參考前幾筆資料(的平均值)來建立楊氏模數
